Quran Hifz Tracker

A Google Sheets application and mobile-first web app that tracks a child's Qur'an memorization through the three-stage memory pipeline — Sabaq, Para Sabaq, Manzil — against two milestones: Memorized and Recited to Hafiz.

This is a seven-part documentation set. Each page is self-contained and cross-linked. Read the Product Spec to understand what the product is and rebuild it from scratch; the Architecture for how it's structured; the Code Structure for the file/module map; the User Guide for day-to-day operation (both regular users and the owner/admin); the Dev Guide for working on the repo (the clasp loop, the staging→prod pipeline, tests and AI agents); the Deployment page for what's deployed and where config & secrets live; and the Migration runbook for moving the live instance to repo-driven config.

The product at a glance

The mobile web app has four nav tabs — Home (headline stats, dual finish projections, recited-gap warning), Progress (zoomable memory-state map: Juz/Surah overview → page strip → line detail), Trends (sabaq/activity/completion charts; +/− grain zoom Day↔Quarter; onboarding & coverage-sweep entries kept off the activity bars), and Log (the four logging workflows). Four header tools sit alongside the nav tabs: ✎ spatial position picker, ⚙ Settings (edition, memory-model params), 🕐 History time machine (read-only as-of / compare view across any past dates), and 💬 Ask AI chatbot (grounded natural-language Q&A over your own data — never invents numbers). Verse (āyah) progress is shown throughout as "N / 6,236 āyāt" and the spatial picker accepts both physical (Juz→Page→Line) and semantic (Surah:Āyah) addressing.

Key vocabulary (one line each)

TermMeaning
HifzMemorization of the Qur'an — the thing being tracked.
HafizThe teacher who has memorized the whole Qur'an; the student recites to them for verification.
SabaqNew lines memorized each day (short-term in).
Para Sabaq / SabqiRevision of the most recent pages — consolidates short-term → long-term.
ManzilRotating revision of everything already in long-term memory (keep it in).
MemorizedMilestone 1: the student claims a line is done (self-reported, provisional).
Recited to HafizMilestone 2: the teacher has verified the line by hearing it (authoritative).
LineThe unit of tracking. The Madinah Mushaf has 8,820 lines across 604 pages.
MushafThe printed edition learned from (Madinah, editions V1/V2).
Spatial pickerThe verse-aware position/portion picker — a page-strip ⇄ line list, addressable by Juz→Page→Line or Surah:Āyah.
Verse-snapWhen logging a portion, the app rounds the selection outward to whole āyāt by default (raw-line opt-out available).
History / time machineRead-only view (🕐 header icon): see the full app state as of any past date, or compare two dates side-by-side.
